# Marketing assets

Off-product brand assets that get uploaded to external surfaces (storefronts,
checkouts, social), not served by the web app. Web-served brand rasters live in
`apps/web/public/` and are generated by `scripts/gen-brand-icons.py`.

| File                     | Where it's used                                                      | Regenerate                        |
| ------------------------ | -------------------------------------------------------------------- | --------------------------------- |
| `lemonsqueezy-cover.png` | Lemon Squeezy product media (managed-plan checkout), 4:3 `1600×1200` | `python3 scripts/gen-ls-cover.py` |
